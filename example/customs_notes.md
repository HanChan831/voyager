# 冰岛入境须知与当地行为规范

## 一、药品入境规定

### ✅ 哮喘药（处方吸入剂）

- 哮喘吸入剂属于普通处方药，**允许携带入境**
- 从非EEA国家（含中国）入境：最多可携带 **100天用量**
- **必须携带以下任一证明**：
  1. 医生出具的英文说明信（含姓名、药名、用法用量）
  2. 原始处方（含姓名、药名、用法用量）
  3. 药店标签/贴纸（贴在药品原包装上）
- 建议：**保留原包装+药店标签+英文医生说明**，随身携带不要托运
- 吸入剂不受登机液体限制，可直接随身携带

### 一般药品建议

- 所有自用药保留原包装
- 携带英文版处方或医生说明信
- 放随身行李，不要托运

## 二、入境禁带物品

| 类别 | 说明 |
|------|------|
| 肉类/奶制品/生鲜 | 未经加工的肉、禽、蛋、奶制品严禁携带 |
| 动植物制品 | 未经许可的动植物及其制品 |
| 武器 | 匕首(>12cm)、弹簧刀、十字弓、手铐等 |
| 毒品/麻醉品 | 严禁 |
| 二手骑马装备 | 需消毒证明 |

> ⚠️ 与你相关的提醒：如想带肉类零食（牛肉干、鸭脖等），禁止入境。泡面/饼干等加工食品一般没问题。

## 三、当地行为规范

### 小费
- 冰岛 **没有小费文化**，不需要给
- 账单已含服务费，强行给小费可能被视为侮辱
- 表达感谢：说"Takk"（谢谢）+ 真诚微笑即可

### 温泉礼仪
- **入池前必须裸身淋浴**，这是最严格的文化要求
- 保持安静、不跳入池中

### 拍照
- 走标记步道，不为了拍照踩苔藓或封闭区域
- 苔藓需数十年恢复，踩踏是永久破坏
- 很多景点禁止无人机，注意标示
- 拍当地人先征得同意

### 社交
- 握手礼，直接坦诚的沟通风格
- 守时（迟到5分钟即视为不尊重）
- 直呼名字是常态
- 冰岛自来水可直接饮用

### 环境保护
- 严禁越野驾驶（off-road），重罚
- 不带走石头/植物
- 只在指定营地露营

---

## 四、支付

- 信用卡/借记卡几乎全覆盖，几乎不需现金
- 建议带一张芯片信用卡（加油需要4位PIN码）
