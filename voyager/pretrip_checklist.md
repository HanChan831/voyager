# 行前准备清单汇总指南

第9步要产出一份**独立的、汇总性的"行前准备清单"**，跟签证板块、本地特殊需求板块是两回事——那两块是"分类详细说明"，这一块是"把所有需要在出发前做的事，按时间紧迫度汇总成一份用户能直接打钩执行的清单"，并行存在、互相呼应，不要互相替代。

## 清单来源：三部分拼起来

1. **第2步签证核查里产生的行动项**：如果需要办签证，把"准备材料""预约递交""等待审理"这些步骤里**用户需要亲自动手**的部分摘出来（比如"去拍证件照""准备资金证明"），不要照搬整段说明文字。
2. **第5步本地特殊需求里产生的行动项**：比如租车需要"办国际驾驶许可证"，这种本身就是行前要做的事。
3. **通用行前准备项**：参考下面的菜单，根据这趟行程的特点（国内/出境、停留天数、季节、是否自驾等）挑选适用的几项，不适用的不要硬塞。

## 通用行前准备项菜单

按"要不要做"的判断依据列出来，不是每趟行程都要全部用上：

- **证件类**：检查护照剩余有效期（出境游通用，国内出行不需要）、护照空白页是否够用、身份证是否在有效期内（国内出行）、复印/拍照备份重要证件
- **资金与支付**：兑换目的地货币（出境游，国内不需要）、确认信用卡/移动支付在当地是否可用、通知发卡行"即将出境刷卡"避免被风控冻结（出境游）
- **通讯**：购买当地 SIM 卡或开通国际漫游/eSIM（出境游）、下载离线地图（信号不好的目的地建议）
- **预订确认**：确认机票/火车票电子凭证、确认酒店预订单、确认需要预约的景点门票（比如有些场馆需要提前线上预约才能入场）
- **健康与安全**：常备药品（晕车药、肠胃药等，按个人情况）、了解目的地是否有特殊疫苗/健康申报要求（出境游，呼应第2步签证核查里查到的信息）、购买旅行保险
- **行李与装备**：根据目的地天气/季节准备衣物（联网查一下出行那几天的天气预报或当季气候特点）、电源适配器（如果目的地插座/电压制式不同）、第5步涉及的特殊装备（潜水/滑雪等）
- **出发前最后确认**：行程单/订单截图保存到手机本地（以防出发后没网）、告知家人/朋友行程安排、确认交通方式去机场/车站的时间留有余量

## 怎么决定"国内出行"该列哪些

国内出行通常可以跳过：护照有效期、货币兑换、SIM卡/漫游、电源适配器、银行出境提醒这几项；保留：身份证有效期、预订确认、天气准备、健康准备里的常备药品、出发前最后确认这几项。不要把出境游专属的条目原样塞进国内行程的清单里，显得不专业。

## 写入 `pretrip_checklist.md` 的建议结构

```markdown
## 行前准备清单

- [ ] 检查护照有效期（需≥6个月）
- [ ] 准备签证申请材料并预约递交
- [ ] 购买旅行保险
- [ ] 兑换日元现金（建议5000元等值起）
- [ ] 开通国际漫游或购买当地SIM卡
- [ ] 下载离线地图
- [ ] 确认大熊猫基地门票已线上预约
- [ ] 查看出行当周天气，准备相应衣物
```

按**时间紧迫度从上到下排序**（越早该做的排越前面，比如签证类通常最紧迫），不要按"签证类/资金类"这种分类硬分段——用户要的是"一份能从上往下打勾执行的清单"，不是一份分类说明文档（分类说明已经在签证板块和本地需求板块里有了）。

## 在最终网页里怎么呈现

这份清单在 `trip_template.html` 里有专门的"行前准备"标签页，每一项会渲染成可以真实点击打勾的复选框（不是纯文字的"☐"符号），具体写法见 `html_content_guide.md`。
